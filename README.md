# 🚀 CareerBridge Pro – Job Portal Web Application

CareerBridge Pro is a modern, responsive, multi-page job portal web application designed to simulate a real-world recruitment platform. It helps users explore job opportunities, apply for jobs, save listings, and track their applications through a dynamic dashboard.

This project is built using **HTML, CSS, and JavaScript (Vanilla JS)** and focuses on frontend development, UI/UX design, and client-side data handling using LocalStorage.

## 📌 Project Overview

The main objective of this project is to build a clean, interactive, and user-friendly job portal that demonstrates core frontend development skills. It replicates basic functionalities of real job platforms like:

- Job browsing
- Application tracking
- Saved jobs system
- User authentication UI (frontend only)

---

## ✨ Key Features

### 🏠 Home Page
- Modern landing page with hero section
- Job search UI (title + location input)
- Featured job listings
- Call-to-action section

### 💼 Jobs Page
- List of available jobs
- Apply & Save job functionality
- Clean and responsive job cards

### 📊 Dashboard
- Displays:
  - Total applied jobs
  - Saved jobs count
- Dynamic job tracking using LocalStorage
- Applied & saved job sections

### 🔐 Authentication Pages
- Login page UI
- Signup page UI
- Form validation using JavaScript

### 📄 Additional Pages
- About page with company vision
- Contact page with form & contact details

## 🛠 Tech Stack Used

- CSS3 – Styling and responsive design
- JavaScript (ES6) – Functionality and interactivity
- LocalStorage – Data persistence (frontend storage)
